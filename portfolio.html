Responder

Get the .apk

Click here

This application is still in testing and the API Key is for production, so the GPS services may not work in THIS .apk, also, information contained within may be propietary.

Responder is an Android application that allows first responders to collect information and GPS coordinates for victims of disasters. Responder also provides first response guides for general purposes as well as for providing first response to individuals with access and functional needs.

Features:

SQLite database storage and access

Form collection

GPS location services

The ability to email data, including KML files for use with Google Earth.

Guides written in HTML



APSU GIS Standard Libs

APSU GIS Standard Libs is an Android library which contains many standard classes and functions for use with Android applications. The APSU GIS Center intends to use this library to standardize its future building of Android applications.

Features:

Class for creating http request objects

Methods for processing http requests

Methods to parse JSON objects

Login method to authenticate with server API

Class for mapping JSON objects to desired application objects



Sketch

An independent group study that uses an Android device to take a picture, convert it to a vector image, apply filters, and print it on a plotter machine.

Features:

Ability to capture images using device's default camera

Store images into device's default gallery

View and select images in gallery

Scale and convert image into vector image using GCode and send to Arduino to interpret and draw image on a plotter device

Asset Management

View or update server info
